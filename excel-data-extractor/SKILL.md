# Excel 数据提取技能

根据用户要求，从Excel文件中智能提取和汇总数据。

## 核心能力

- **自然语言查询** — 用户用口语描述需求，自动解析并提取数据
- **字段筛选** — 按列名提取指定字段
- **条件过滤** — 按条件筛选行数据
- **数据汇总** — 求和、平均值、计数等统计计算
- **多表处理** — 支持多Sheet和跨表数据整合

## 工作流程

### 1. 定位数据源

用户可能提供：
- 完整文件路径（如 `D:/编程/补贴数据.xlsx`）
- 部分路径关键词（如"补贴数据""寨隆镇"）
- 未提供，需搜索定位

**未提供路径时**，按以下顺序查找：
```
D:/编程/ 目录下的所有 .xlsx/.xls 文件
工作目录下的 .xlsx/.xls 文件
用户桌面/下载目录
```

### 2. 分析表结构

使用 openpyxl 或 pandas 读取Excel：
```python
import pandas as pd

# 读取Excel（自动检测编码和表头）
df = pd.read_excel(file_path, header=0)

# 显示基本信息
print(f"总行数: {len(df)}, 总列数: {len(df.columns)}")
print(f"列名: {list(df.columns)}")
print(df.head(10))  # 前10行预览
```

### 3. 解析用户需求

将自然语言转换为数据操作：

| 用户需求 | 数据操作 |
|---------|---------|
| "提取姓名和补贴面积" | 选择列 |
| "找出面积大于5亩的农户" | 条件筛选 |
| "计算总补贴金额" | 求和统计 |
| "按村组汇总" | 分组聚合 |
| "导出到新文件" | 保存结果 |

### 4. 执行数据提取

```python
# 基础提取
result = df[['农户姓名', '补贴面积']].copy()

# 条件筛选
result = df[df['补贴面积'] > 5]

# 分组汇总
summary = df.groupby('村组')['补贴面积'].sum()

# 复杂条件（用户描述转SQL-like逻辑）
mask = (df['补贴面积'] > threshold) & (df['村组'] == village)
result = df[mask]
```

### 5. 返回结果

- **简单结果**：直接展示表格数据
- **大量数据**：导出为Excel文件
- **统计结果**：展示汇总数据和计算逻辑

## 数据验证

执行前验证：
- 文件是否存在
- 列名是否匹配（处理模糊匹配，如"姓名"≈"农户姓名"）
- 数据类型是否正确

异常处理：
- 文件被占用 → 提示关闭后重试
- 列不存在 → 列出可用列名供用户确认
- 编码问题 → 尝试 GBK/UTF-8 编码切换

## 输出格式

| 场景 | 输出方式 |
|-----|---------|
| < 20行数据 | 表格展示 |
| 20-100行数据 | 表格 + 文件路径 |
| > 100行数据 | 仅文件，附带统计摘要 |
| 统计结果 | 卡片展示计算过程和结果 |

## 常用模板

### 农业补贴数据模板

```python
# 补贴核验表提取
补贴字段 = ['序号', '姓名', '身份证号', '补贴面积', '补贴金额', '银行卡号']

# 按农户提取
农户数据 = df[df['姓名'] == 农户名][['姓名', '补贴面积', '补贴金额']]

# 面积核验：合计 = sum(各丘块面积)
合计面积 = df['丘块面积'].sum()
```

## 注意事项

- **原始数据保留**：不修改源文件，只读操作
- **小数精度**：面积保留2位小数，金额保留2位小数
- **空值处理**：显示为空值，不替换为0
- **编码兼容**：Windows中文环境优先GBK编码

## 配套脚本

### `scripts/extract_data.py`
Excel数据提取交互工具，双击 `启动提取工具.bat` 运行。
支持序号选择与字段名输入两种模式，输出Excel/JSON/CSV三格式。

### `scripts/sensenova_vision.py`
SenseNova Vision API 调用脚本，识别图片内容。

```bash
# 基本使用（输出JSON）
python scripts/sensenova_vision.py 图片路径.jpg

# 自定义提示词
python scripts/sensenova_vision.py 图片路径.png --prompt "提取表格中的所有数据"

# 只输出文本（方便管道处理）
python scripts/sensenova_vision.py 图片路径.jpg --text
```

API密钥通过环境变量配置：
- `SN_VISION_API_KEY`（优先）或 `SN_API_KEY`
- `SN_VISION_MODEL`（默认 sensenova-6.7-flash-lite）
- `SN_VISION_BASE_URL`（默认 https://token.sensenova.cn/v1）
